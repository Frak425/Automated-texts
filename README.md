# Appointment Reminder System

An automated reminder system that uses google calendar, google sheets, and twilio to send automated SMS messages to clients, reminding them of both upcoming appointments and general information

## Features

- Reads appointments from google calendar
- Retrieves information from google sheets
- Matches appointments to contact info using email as the key
- Sends configurable SMS reminders

## Tech Stack

- Python 3.14
- Google Calendar API
- Google Sheets API
- Twilio
- OAuth 2.0

## Installation

1. Clone the repository
2. Create a virtual environment
3. Install dependencies with '''pip install -r requirements.txt'''
4. Create a `.env` file
5. Add Google credentials

## Usage
